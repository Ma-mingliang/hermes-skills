# 系统分析（横向+纵向模式）

## 横向模式（倒立摆模型）

### 状态空间模型
状态变量：x = [θ, θ̇]ᵀ（俯仰角、俯仰角速度）
输入：u = 电机电流

A = [0,    1   ]
    [g/H,  0   ]

B = [0          ]
    [-(L+Lg)/H  ]

### 传递函数
G(s) = -(L+Lg) / (H·s² - g)

代入数值（H=0.63m, L=1.17m, Lg=0.153m）：
G(s) = -2.10 / (s² - 15.56)

### 极点
p1 = +√(g/H) = +3.94（不稳定）
p2 = -√(g/H) = -3.94（稳定）

### 可控性
可控性矩阵 C = [B, AB]
det(C) = -4.41 ≠ 0
→ 系统完全可控

### 物理含义
- 开环不稳定（正实数极点）
- 需要主动控制才能保持平衡
- 自然频率 ωn = √(g/H) = 3.94 rad/s
- 阻尼比 ζ = 0（无阻尼项）

## 纵向模式（自行车模型）

### 状态空间模型
状态变量：x = [φ, φ̇, δ]ᵀ（侧倾角、侧倾角速度、前轮实际转角）
输入：u = δc（舵机命令角）

A = [0,      1,      0    ]
    [-15.430, 0,      6.705]
    [0,      0,     -20   ]

B = [0]
    [0]
    [20]

### 传递函数
G(s) = b / (τs·s³ + s² - a·s - a/τs + b/τs)

代入数值：
G(s) = 6.705 / (0.05s³ + s² - 15.43s - 174.49)

### 极点
p1 = -20.34（稳定，舵机动态）
p2 = +3.10（不稳定，侧倾不稳定）
p3 = -2.77（稳定）

### 可控性
可控性矩阵 C = [B, AB, A²B]
det(C) = -359705.72 ≠ 0
rank(C) = 3 = 系统阶数
→ 系统完全可控

可控性条件：1/τs ≠ 0（舵机一阶模型保证）且 b ≠ 0（要求v > 0）

### 闭环极点问题
使用论文增益 K = [-5.313, -1.352, 0.379] 时：
- 连续闭环极点中有一个正实部（不稳定）
- 使用相反符号 K = [5.313, 1.352, -0.379] 时所有极点稳定

原因：论文使用离散LQR（T_s=0.02s），增益针对离散系统设计

## 系统参数（霍昱达论文）

### 横向模式
- H = 0.63 m（车身高度）
- L = 1.17 m（轴距）
- Lg = 0.153 m（重心偏移）
- g = 9.81 m/s²

### 纵向模式
- m = 23.3 kg（整车质量）
- h = 0.144 m（重心高度）
- L = 0.76 m（轴距）
- I_phi = 1.65 kg·m²（侧倾转动惯量）
- J_phi = 2.133 kg·m²（等效转动惯量）
- a = mgh/J_phi = 15.430 s⁻²
- b = mhv²/(J_phi·L) = 6.705 s⁻²（v=1.8 m/s）
- τs = 0.05 s（舵机时间常数）
