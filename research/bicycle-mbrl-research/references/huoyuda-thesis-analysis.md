## 硬件平台（第2章）

### 执行器
- 轮毂电机 × 2（两端支撑臂底部，与轮毂集成一体）
- 180°无刷舵机 × 2（锤子品牌，PWM周期20ms，脉宽0.5-2.5ms，最大扭矩70kg·cm，响应时间0.082s）
  - 模式切换时驱动轮组旋转90°
  - 纵向模式下在90°基准附近小角度偏转转向

### 传感器
- WHEELTEC N100惯导模块（串口，100Hz输出）
  - 安装在车架中部靠近质心位置
  - 输出：俯仰角、俯仰角速度、偏航角速度

### 主控
- Arduino Nano ESP32（ESP32-S3，主频240MHz，BLE+WiFi）
- 自制PCB载板（集成SN65HVD230 CAN收发器）

### 驱动器
- 布瑞特BRTFOCCAN200驱动器（12-80V，持续5A，最大10A）
- 支持电流环/速度环/位置环/通信控制

### 供电
- 动力：48V锂电池组 → 布瑞特驱动器 → 轮毂电机
- 外设：22.2V 6S航模锂电池 → 8.4V（舵机）+ 5V（主控/IMU）

### 通信
- 微信小程序 "Bike BLE"（BLE通道）
- Idle/Segway/Bike三种模式切换
- 实时遥测：姿态角、电池电压

## 横向模式建模（第3章）

### 模型假设
- 简化为单自由度倒立摆
- 忽略车架弹性变形和左右轮差速转向
- 轮胎接触作用合并到等效控制输入

### 动力学方程
```
Jθ̈ = mgh sin θ - cθ̇ + ku(t)    (公式3.4)
```
- θ：俯仰角（向x轴负方向倾斜为正）
- h：等效质心到轮轴点距离
- Jθ：车体绕轮轴点的等效转动惯量
- m：车体等效质量
- c：等效阻尼系数
- k：控制输入到俯仰力矩的等效系数

### 线性化（小角度近似）
```
Jθ̈ + cθ̇ - mghθ = ku(t)    (公式3.6)
```

### 传递函数
```
G(s) = θ(s)/U(s) = -k / (Js² + cs - mgh)    (公式3.7)
```
- 分母含-mgh项，反映负刚度特征（开环不稳定）

### 控制方法：串级PID
- 速度外环（PI）：目标速度vs实际速度 → 目标俯仰角补偿
- 直立内环（PD）：目标俯仰角vs实际俯仰角 → 电机电流命令
- 电流整形：限幅、死区补偿、启动电流补偿

### 关键参数
| 参数 | 作用 |
|------|------|
| Kpθ | 直立内环比例系数 |
| Kdθ | 直立内环微分系数 |
| Δθ | 目标平衡角人工修正量 |
| Kpv | 速度外环比例系数 |
| Kiv | 速度外环积分系数 |
| Smax | 速度外环积分限幅 |
| α | 平均速度反馈低通滤波系数 |
| Imax | 电机电流命令限幅 |
| Is | 最小启动电流 |
| Id | 小电流死区阈值 |

### 实验结果
- 稳态俯仰角标准差：约0.16°
- 约88%采样点在±0.5°带内

## 纵向模式建模（第4章）

### 模型假设
- 低阶侧倾-转向模型（不采用完整Whipple模型）
- 忽略偏航自由度的二阶动力学
- 车架弹性变形与轮胎侧偏合并到未建模扰动
- 纵向速度v在控制设计阶段视为已知常量（v=1.8m/s）
- 舵机用一阶位置跟踪环节描述

### 物理参数
| 参数 | 值 | 含义 |
|------|-----|------|
| m | 23.3 kg | 整车总质量 |
| g | 9.81 m/s² | 重力加速度 |
| h | 0.144 m | 重心到地面接触线距离 |
| L | 0.76 m | 轴距 |
| v | 1.8 m/s | 纵向速度 |
| Iφ | 1.65 kg·m² | 绕整车重心的侧倾转动惯量 |
| τs | 0.05 s | 舵机等效时间常数 |

### 等效转动惯量（公式4.1）
```
Jφ = Iφ + mh² = 1.65 + 23.3 × 0.144² = 2.133 kg·m²
```

### 侧倾动力学方程（公式4.6）
```
Jφφ̈ = -mghφ + mh²v²δ/L
```
- 第一项：重力倾倒力矩（不稳定）
- 第二项：转向扶正力矩（需v>0）

### 归一化形式（公式4.7）
```
φ̈ = -aφ + bδ
```
- a = mgh/Jφ = 15.430 s⁻²（重力正反馈）
- b = mhv²/(JφL) = 6.705 s⁻²（转向扶正能力）

### 舵机模型（公式4.8）
```
δ̇ = (δc - δ) / τs
```
- δc：命令角
- δ：实际转角
- τs = 0.05 s

### 三阶状态空间模型（公式4.9-4.11）
状态变量：x = [φ, φ̇, δ]ᵀ
输入：u = δc

```
A = [0,      1,      0    ]
    [-a,     0,      b    ]
    [0,      0,    -1/τs  ]

B = [0]
    [0]
    [1/τs]
```

代入数值：
```
A = [0,       1,       0   ]
    [-15.430, 0,       6.705]
    [0,       0,     -20   ]

B = [0]
    [0]
    [20]
```

### 传递函数（δc → φ）
```
G(s) = b / (τs·s³ + s² - a·τs·s - a + b)
     = 6.705 / (0.05s³ + s² - 0.7715s + 15.43)
```

### 可控性分析
可控性矩阵：C = [B, AB, A²B]
- rank(C) = 3（满秩）
- det(C) = -359705.72 ≠ 0
- **系统完全可控**

可控性条件：1/τs ≠ 0 且 b ≠ 0
- 前者由舵机一阶模型保证
- 后者要求 v > 0（零速下不可控！）

### 开环极点
- p1 = -20.34（稳定，舵机极点）
- p2 = +3.10（不稳定，侧倾不稳定）
- p3 = -2.77（稳定）

### LQR设计

#### 权重选择（Bryson规则）
归一化参考量：
- φmax = 5° = 0.0873 rad
- φ̇max = 30°/s = 0.524 rad/s
- δmax = 25° = 0.436 rad
- δc,max = 25° = 0.436 rad

最终权重（比Bryson初值更保守）：
```
Q = diag(32.83, 0.365, 1.31)
R = 630.3
```

#### 反馈增益（公式4.20）
```
K = [-5.313, -1.352, 0.379]
```

#### 控制律（公式4.21）
```
δc = -K·x = 5.313φ + 1.352φ̇ - 0.379δ
```

- K0 = 5.313：侧倾角误差驱动的转向恢复强度
- K1 = 1.352：侧倾角速度阻尼
- K2 = -0.379：抑制前轮转角通道耦合

### 闭环极点
- 所有特征值模值 < 1（单位圆内）
- 系统渐近稳定

### 实验结果
- 标称速度稳态：95.7%侧倾角样本在±1°带内
- 全程在±2°带内
- 4.92°初始侧倾角扰动下，约1.92s进入±2°带
- 收敛时间约1.26s（仿真）

## 与横向模式对比

| 特性 | 横向模式（Segway） | 纵向模式（Bike） |
|------|-------------------|-----------------|
| 构型 | 两轮左右布置 | 前后轮纵向排列 |
| 系统阶数 | 二阶 | 三阶 |
| 状态变量 | [θ, θ̇] | [φ, φ̇, δ] |
| 控制方法 | 串级PID | LQR |
| 扶正机理 | 电机直接产生力矩 | 转向产生侧向加速度 |
| 速度依赖 | 无 | 需要v > 0 |
| 控制频率 | 100Hz | 50Hz |
| 输出 | 电机电流 | 舵机角度 |

## 关键公式汇总

1. 横向传递函数：G(s) = -k / (Js² + cs - mgh)
2. 纵向传递函数：G(s) = b / (τs·s³ + s² - a·τs·s - a + b)
3. 转向几何：κ ≈ δ/L, ψ̇ ≈ vδ/L
4. 侧向加速度：ay ≈ v²δ/L
5. 等效转动惯量：Jφ = Iφ + mh²
