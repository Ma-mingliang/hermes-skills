## 电机系统辨识（重要方法论）

### PWM-转速关系
实验测得线性关系：`n = 0.156 * q`（n=eRPM, q=PWM占空比%）

### 电机传递函数辨识
方法：在占空比10%时施加20%阶跃扰动，记录响应曲线。

模型：一阶惯性环节 `G(s) = K / (τs + 1)`

辨识结果：`G(s) = 1.6 / (0.301s + 1)`

### 两点法参数辨识
```
τ = (t₂ - t₁) / ln[(1 - y(t₁)) / (1 - y(t₂))]
其中 y(t₁) = 0.632, y(t₂) = 0.393
K = Δy / Δu
```

验证指标：
- t*₃ = 2τ, y(2τ) ≈ 0.87
- t*₄ = 2.5τ, y(2.5τ) ≈ 0.39
- t*₅ = 1.44τ, y(1.44τ) ≈ 0.50

状态空间：

A = [0, 1; g/H, 0]    B = [0; -(L+Lg)/H]

## 数据测量方法

### IMU数据
- WHEELTEC N100，100Hz输出
- 输出：俯仰角、俯仰角速度、偏航角速度
- 通信：UART串口

### 数据收集方式
1. BLE实时传输（霍昱达方案）：12字段调试帧
2. SD卡记录：大量数据，离线分析
3. CAN总线记录：电机状态数据

### 信号优先级（高佳旗定义）
| 优先级 | 信号 |
|--------|------|
| 0 | 遥控器无线信号 |
| 1 | IMU姿态数据 |
| 2 | 驱动器霍尔信号 |
| 3 | 舵机PWM信号 |
| 4 | 驱动器PWM信号 |

## 系统辨识实验方案（综合两篇论文）

### 实验1：电机辨识
- 输入：阶跃PWM信号
- 输出：电机转速
- 辨识：一阶惯性环节 K/(τs+1)

### 实验2：横向模式辨识
- 输入：阶跃电流
- 输出：俯仰角响应
- 辨识：倒立摆模型参数 (m, h, J, c, k)

### 实验3：纵向模式辨识
- 输入：阶跃转向角
- 输出：侧倾角响应
- 辨识：自行车模型参数 (a, b, τs)

### 数据记录格式（CSV）
```
时间戳, 俯仰角, 俯仰角速度, 侧倾角, 侧倾角速度,
前轮转角, 电机1转速, 电机2转速, 舵机命令, 电机电流
```
