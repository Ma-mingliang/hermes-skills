## 阶段2：系统辨识数据收集

### 2.1 电机系统辨识

**目标**：获取电机传递函数 G(s) = K / (τs + 1)

**方法**：两点法（参考高佳旗论文）

**步骤**：
1. 电机空载运行
2. 施加阶跃PWM信号（0% → 10%）
3. 记录转速响应（100Hz，5秒）
4. 重复3次取平均

**两点法公式**：
- y(t₁) = 0.632, y(t₂) = 0.393
- τ = (t₂ - t₁) / ln[(1-y(t₁))/(1-y(t₂))]
- K = Δy / Δu

### 2.2 横向模式系统辨识

**目标**：辨识倒立摆模型 G(s) = -k / (Js² + cs - mgh)

**步骤**：
1. 车体固定在横向模式
2. 人工扶持到平衡位置附近
3. 施加阶跃电流（500mA）
4. 快速释放，记录俯仰角响应
5. 100Hz采样，5秒，重复5次

**数据记录**：时间戳、俯仰角θ、俯仰角速度θ̇、电机电流u

### 2.3 纵向模式系统辨识

**目标**：辨识状态空间模型 ẋ = Ax + Bu（x = [φ, φ̇, δ]ᵀ）

**步骤**：
1. 恒定速度行驶（1.5 m/s）
2. 施加阶跃转向命令（δc = 10°）
3. 记录侧倾角响应
4. 100Hz采样，3秒，重复5次
5. 更换速度（1.0, 2.0 m/s）重复

**数据记录**：时间戳、侧倾角φ、侧倾角速度φ̇、舵机命令δc、车速v

### 2.4 数据收集工具

**微信小程序数据记录功能**（2026-05-30已实现）：
- 在微信小程序中点击"开始记录"按钮
- 进行测试（切换模式、调整参数等）
- 点击"停止记录"按钮
- 点击"导出CSV"按钮，通过微信分享保存文件

**CSV格式**：timestamp, modeId, roll, pitch, batt

**Python分析脚本**：`tools/analyze_data.py`
```bash
python tools/analyze_data.py bike_data.csv
```
