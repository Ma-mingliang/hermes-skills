# Hermes Skill Factory

A meta-skill plugin for Nous Research's Hermes AI agent that watches your workflows and automatically turns them into reusable skills.

## Stars: ⭐339

## URL: https://github.com/Romanescu11/hermes-skill-factory

## What is Hermes Skill Factory?

Hermes Skill Factory is a meta-skill that automatically captures your workflows and converts them into reusable skills. It solves the problem of:
- Workflows disappearing after sessions
- Having to explain the same workflow again
- Manual skill creation overhead

## Why use Hermes Skill Factory?

This skill helps you:
- Automatically capture workflows
- Convert workflows to reusable skills
- Build your skill library over time
- Save time on repetitive tasks

## Key Features

1. **Workflow Capture**: Automatically watches and records your workflows
2. **Skill Generation**: Converts workflows into reusable skill templates
3. **Pattern Recognition**: Identifies common patterns in your work
4. **Skill Library**: Builds a growing library of your personal skills

## How to use

1. Install the skill factory plugin
2. Work normally with Hermes Agent
3. The factory automatically captures your workflows
4. Review and refine generated skills
5. Use the skills in future sessions
